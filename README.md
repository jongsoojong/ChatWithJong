# ChatWithJong
A full-stack Chat app

## Getting Started

### Prerequisites

1. Clone the repository to your computer.

2. Run npm install to install the dependencies required for the app.

3. Run npm start to start the server.

4. Go to localhost:8888.

###Built With

- Mongoose
- Express
- Angular
- Nodejs
